# Frontend

Frontend made with Angular



## Features

- Simple components
- Used ts interfaces
- Full responsive


## Installation

Install the dependencies using npm

```bash
  npm install
```

Run locally

```bash
  npm run start
```
## Running Tests

To run tests, run the following command

```bash
  npm run test
```


## Tech Stack

 - Angular as web framework
 - HTML and CSS to build simple and functional interface


## Routes

 - `/admin` - list requests with filters
 - `/judge` - list pending requests
 - `/judge/:id` - aprove/reject a request
 - `/create` - create a request
